# Aesthetic Foods

Aesthetic Foods is a modern fast-food takeaway restaurant website designed for the Pakistani market. The platform helps customers easily browse menus, explore deals, customize meals, and place takeaway orders through a clean and mobile-friendly interface.

The project solves common restaurant challenges such as manual ordering, poor online presence, and reliance on third-party delivery platforms. By providing a dedicated ordering platform, restaurants can improve customer experience, increase direct sales, and manage orders more efficiently.

### Tech Stack

* React.js
* TypeScript
* Tailwind CSS
* Firebase Authentication
* Cloud Firestore
* Firebase Storage
* Firebase Hosting
* Cloud Functions
* Cloudinary (image optimization)

### Key Features

* Interactive food menu
* Food search and filtering
* Meal customization
* Cart and checkout system
* Promotional deals and offers
* WhatsApp ordering integration
* Responsive mobile-first design
* Admin dashboard for menu and order management

Aesthetic Foods focuses on creating a fast, visually appealing, and user-friendly food ordering experience tailored to local customer behavior and restaurant business needs.
